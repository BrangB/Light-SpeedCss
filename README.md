# @brang/light-speedcss

A lightning-fast, utility-first CSS framework for building modern, responsive, and beautiful user interfaces with minimal effort.

## üöÄ Features

- üé® Utility-first classes for layout, typography, spacing, and more  
- ‚ö° Lightweight and performance-focused  
- üß© Easily customizable with theming support  
- üì± Responsive design baked in  
- üåò Dark mode ready

## üì¶ Installation

```sh
npm install light-speedcss
```

## üîß Usage
Use By Linking the files from node_modules:
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@brang/light-speedcss@latest/dist/light-speed.css" />
```

```html
<div class="text-center bg-primary text-white p-4 rounded">
  Welcome to Light-SpeedCSS üöÄ
</div>
```

## üìè Spacing Utilities

Control layout spacing using margin, padding, and gap utilities.

### üî≤ Margin & Padding Scale

| Class | rem | px  |
|-------|-----|-----|
| `*-0`   | 0   | 0   |
| `*-1`   | 0.25rem | 4px |
| `*-2`   | 0.5rem  | 8px |
| `*-3`   | 1rem    | 16px |
| `*-4`   | 1.5rem  | 24px |
| `*-5`   | 2rem    | 32px |
| `*-6`   | 2.5rem  | 40px |
| `*-8`   | 3rem    | 48px |
| `*-10`  | 4rem    | 64px |

### ‚ÜîÔ∏è Margin

- `.m-*` ‚Äì All sides  
- `.mx-*` ‚Äì Horizontal (left + right)  
- `.my-*` ‚Äì Vertical (top + bottom)  
- `.mt-*`, `.mr-*`, `.mb-*`, `.ml-*` ‚Äì Individual sides

### üì¶ Padding

- `.p-*` ‚Äì All sides  
- `.px-*` ‚Äì Horizontal  
- `.py-*` ‚Äì Vertical  
- `.pt-*`, `.pr-*`, `.pb-*`, `.pl-*` ‚Äì Individual sides

### üî≥ Gap (Flex & Grid)

- `.gap-*` ‚Äì Sets gap between items  
- `.row-gap-*` ‚Äì Vertical gaps only  
- `.col-gap-*` ‚Äì Horizontal gaps only

### üß™ Example

```html
<div class="p-4 mb-6">
  <div class="gap-4 flex">
    <div class="p-2 bg-gray-200">Box 1</div>
    <div class="p-2 bg-gray-300">Box 2</div>
  </div>
</div>
```
